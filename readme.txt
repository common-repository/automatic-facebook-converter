The Automatic Facebook Converter automatically renders your site in a Facebook compatible format, in a page/app.

== Description ==

Automatically Convert Wordpress Sites/Blogs into Fan Pages

You don't have to create a new separate site, just to have a Facebook Fan Page. You can simply type in your existing site, into your settings, and this Worpress plugin does the rest!

As soon as it detects your site is within a Facebook page, it will convert it to a compatible format. Your regular site doesn't get affected in any way!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the 'fbac_converter_pms' folder to the '/wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! 

== Frequently Asked Questions ==

= Do I have to create an app to use this? =

No, you don't. You can use it with any Facebook iFrame app.
